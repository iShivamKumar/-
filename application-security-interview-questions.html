Application Security Interview Questions

Some of the questions/topics I was asked during interviews for Application/Product Security Engineer positions. I'm sure this isn't an exhaustive list, but I thought these questions were important to ask and some were difficult to answer. I attempted to include a reference resource for some of the questions/topics; please contact me on Twitter if you have any feedback/suggestions/discussions.

  1. Explain Encryption, Hashing, and Encoding
    • https://www.tutorialspoint.com/cryptography/cryptography_hash_functions.htm
    • https://www.youtube.com/watch?v=b4b8ktEV4Bg
    • https://blog.jscrambler.com/hashing-algorithms/
  2. Explain your approach that you follow while performing pentesting on a web application
  3. Explain DOM XSS. Can DOM XSS be stored? Can CSP Header mitigate DOM based XSS
  4. What is CSRF? Why it occurs? What mitigation strategies you recommend to a developer?
  5. Explain CORS and SOP
  6. What is XXE? Why it occurs and how you'll prevent from it?
  7. Explain different attributes that are used in the Cookie.
  8. Talk about any latest/interesting vulnerability or breach you learnt about
  9. You’ve found xss issue in the source code, what will be your approach for addressing this?
